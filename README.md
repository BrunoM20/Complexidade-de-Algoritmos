# Análise de Desempenho e Complexidade de Algoritmos de Ordenação

Este projeto tem como objetivo analisar e comparar o **desempenho e a complexidade computacional** de diferentes **algoritmos de ordenação**, utilizando abordagens quantitativas e visuais para entender melhor seu comportamento em diferentes cenários.

## Objetivo

Explorar como diferentes algoritmos de busca se comportam em termos de **tempo de execução**, **número de operações** e **complexidade teórica**, considerando conjuntos de dados de tamanhos variados.

## Algoritmos Estudados

- **Bubble Sort (otimizado)**
- **Selection Sort**
- **Quick Sort**
- **Merge Sort**
- **Tim sort: método da linguagem Python - lista.sort()**

## Metodologia

1. **Geração de conjuntos de dados aleatórios**.  
2. **Execução dos algoritmos** em diferentes entradas controladas.  
3. **Coleta de métricas** como:
   - Tempo de execução (em ms)
   - Número de comparações realizadas
   - Complexidade teórica (O(n), O(log n), etc.)
4. **Visualização dos resultados** por meio de gráficos comparativos usando o matplotlib.

## Tecnologias Utilizadas

- **Python 3**
- **Pandas** — Manipulação e análise dos dados  
- **NumPy** — Geração de dados e suporte matemático  
- **Matplotlib** — Visualização dos resultados  
- **Jupyter Notebook** — Para exploração interativa
